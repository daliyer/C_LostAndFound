# 校园失物招领管理系统

一个基于 C 语言开发的控制台应用程序，旨在为校园提供一个高效、便捷的失物招领信息管理解决方案。

## 📝 项目简介

本项目是一个完整的校园失物招领系统，涵盖了失物信息管理的核心功能。系统采用模块化设计，代码结构清晰，便于维护和二次开发。数据通过文件进行持久化存储，确保信息不会因程序关闭而丢失。

## ✨ 主要功能

*   **信息录入**：便捷地添加新的失物信息，包括物品名称、捡拾地点、时间等。
*   **信息查询**：支持按物品名称、捡拾地点等多种条件进行快速检索。
*   **信息修改**：可以更新失物状态（如标记为“已认领”）或修改物品的详细信息。
*   **信息删除**：支持手动删除指定记录，并能自动清理超过30天的过期信息。
*   **数据持久化**：程序退出时自动保存数据，启动时自动加载，保证数据安全。

## 🛠️ 技术栈

*   **编程语言**：C 语言 (C99)
*   **开发环境**：Dev-C++
*   **核心技术**：
    *   结构体与链表/数组
    *   文件 I/O 操作
    *   模块化编程

## 🚀 如何运行

1.  **克隆或下载** 本仓库到本地。
2.  使用 **Dev-C++** 打开项目工程文件 `lostandfound.dev`。
3.  点击“编译并运行”按钮，即可启动程序。
4.  根据控制台菜单提示进行操作。

## 📂 项目结构

*   `main.c`: 主程序入口，负责菜单调度和流程控制。
*   `lost_add.c`: 负责失物信息的录入功能。
*   `lost_search.c`: 负责失物信息的查询功能。
*   `lost_modify.c`: 负责失物信息的修改功能。
*   `file_io.c`: 负责数据的文件读写和系统维护功能。
*   `lostandfound.dev`: Dev-C++ 项目文件。
